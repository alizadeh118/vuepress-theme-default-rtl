قالب VuePress

پیش‌نمایش نسخه راست به چپ قالب پیش فرض

مشاهده بیشتر

عنوان اول

چاپگرها و متون بلکه روزنامه و مجله در ستون و سطرآنچنان که لازم است . امید داشت که تمام و دشواری موجود پایان رسد.

عنوان دوم

تولید سادگی نامفهوم از صنعت چاپ و با استفاده از طراحان گرافیک است. شامل حروفچینی دستاوردهای اصلی.

عنوان سوم

نرم افزارها شناخت بیشتری را برای طراحان رایانه ای علی الخصوص طراحان خلاقی و فرهنگ پیشرو.

# لورم ایپسوم متن

# install
yarn global add vuepress
# OR npm install -g vuepress

# create a markdown file
echo '# Hello VuePress' > README.md

# start writing
vuepress dev

# build to static files
vuepress build
